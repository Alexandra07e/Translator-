TEST 1 - stm: conditia din if trebuie sa fie scalara
Eroare asteptata:
the if condition must be a scalar value

struct A { int x; };

int main() {
    struct A a;
    if (a) {
        return 1;
    }
    return 0;
}


TEST 2 - exprAssign: destinatia trebuie sa fie left-value
Eroare asteptata:
the assign destination must be a left-value

    return 0;
}
